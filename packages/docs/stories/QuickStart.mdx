# Installation

To install Shiba UI, use npm or yarn:

## For React (Web)

```bash
npm install @shiba-ui/browser
```

or

```bash
yarn add @shiba-ui/browser
```

## For React Native (Mobile)

```bash
npm install @shiba-ui/app
```

or

```bash
yarn add @shiba-ui/app
```
