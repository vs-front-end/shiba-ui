# Colors

This section presents the color palette used in the Shiba UI design system. Each color is defined by its hex code and is used to ensure a consistent and cohesive visual identity across the application.

## Browse Colors

Use the search box below to filter colors by name or hex code. Click on any color card to copy its theme reference (e.g., `theme.colors.primary`), or click on the hex code to copy the color value directly.

<ColorsGallery />

## Usage

Use these colors in your components by referencing them from the theme:

```tsx
import { Button } from '@shiba-ui/browser';

function MyComponent() {
  return (
    <>
      <Button text="Primary" background="primary" />
      <Button text="Success" background="success" />
      <Button text="Warning" background="warning" />
      <Button text="Error" background="error" />
    </>
  );
}
```

Or in styled components:

```tsx
import styled from 'styled-components';

const CustomDiv = styled.div`
  background-color: ${({ theme }) => theme.colors.primary};
  color: ${({ theme }) => theme.colors.paper};
`;
```
